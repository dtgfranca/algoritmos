**Quando e onde os arrays podem ser usados**  

* Armazenar e acessar dados sequenciais;
* Armazenar objetos temporariamente;
* Usado por rotinas IO como Buffers;
* Lookup de tabelas e lookup inverso de tabelas;
* Pode ser usado para retornar múltiplos valores vindo de uma funçao;
* Usado em programação dinâmica para cache answers  para subproblemas;
