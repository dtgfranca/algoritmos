** Linked List ** 
Quando usar:  

* Usada em muitas lists, Queue e Stack;
* Ótimo para criar listas circulares;
* Pode ser facilmente modelado objetos do mundo real tal como "trains";
* Usado em "separate chaining", qual  esta presente  certo hastable implementações to deal  
with hashing collisions;
* Frequentemente usado em implementações em adjancy lists para graphs.
